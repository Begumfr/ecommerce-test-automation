# E-commerce Test Automation Project

##  Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven

##  Features
- Login test
- Search product test
- Page Object Model (POM)
- Explicit Wait
- Driver Singleton Pattern
- Config Reader

## How to Run
1. Clone the project
2. Open with IntelliJ
3. Run TestNG tests

##  Test Scenarios
- User can login successfully and check user name displayed
- User can search for products
- Search results are displayed
